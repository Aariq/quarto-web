
## Navbar

Options that define the top navigation bar for a {{< meta project-type >}} For example:

```yaml
---
{{< meta project-type >}}:
  navbar:
    search: true
---
```

::: {#table-navbar}
:::

## Sidebar

Options that define the side navigation area for a website. For example:

```yaml
---
{{< meta project-type >}}:
  sidebar:
    search: true
---
```

::: {#table-sidebar}
:::
